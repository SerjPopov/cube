# 🎲 Кубик - Таймер времени для каждой грани
**Версия: 1.1.2**

Веб-приложение для работы с BLE кубиком, который отслеживает время нахождения на каждой из 6 граней.

## 📋 Описание

Приложение подключается к кубику по Bluetooth Low Energy (BLE) и позволяет:
- Читать текущее время по каждой грани кубика
- Определять, какая грань находится сверху
- Сбрасывать счетчики времени
- Делиться статистикой через мессенджеры или почту

### 🎯 Функциональность

- **🔗 Подключение к кубику** - поиск и соединение по BLE
- **📊 Чтение данных** - получение времени по каждой из 6 граней
- **🔄 Сброс счетчиков** - отправка команды обнуления
- **📤 Поделиться статистикой** - экспорт данных через Web Share API
- **💾 Локальное хранилище** - сохранение данных между сессиями
- **🔍 Отладочные логи** - детальное логирование BLE операций

### 🔌 BLE Спецификации

#### Устройство
- **Имя**: `Cube`
- **UUID устройства**: `85301596-98b5-b0bb-e882-fd4f797fbdb2`

#### Сервис
- **UUID сервиса**: `4FAFC201-1FB5-459E-8FCC-C5C9C331914B`
- **Тип**: Primary Service

#### Характеристика
- **UUID характеристики**: `BEB5483E-36E1-4688-B7F5-EA07361B26A8`
- **Свойства**: Read (0x02), Write (0x08)
- **Формат данных**: Текстовый UTF-8

### 📊 Формат данных

#### Входные данные (от кубика)
```
.     .       Я .       .    bootCount = 1,Я  totalTime = 3s 692ms,  workTime = 3s 673ms,  sleepTime = 19ms.     .     .      .      .       .       .      .       .      currentAge = ЭлмСофтBAK.     .      .      .      .      .      .       .     ОргВопросы = 0 min,      ВладИнтеграции = 0 min,      ТехВопросы = 0 min,      МоиЗадачи = 0 min,      НеРабота = 0 min,      ЭлмСофтBAK = 0 min
```

#### Парсинг данных
- **Текущая грань**: `currentAge = [название]`
- **Время по граням**: `[название] = [число] min`
- **Общее время**: `totalTime = [время]`

#### Команда сброса
- **Формат**: `Uint8Array([1])`
- **Назначение**: Сброс всех счетчиков

## 🚀 Установка и запуск

### 📋 Требования
- **iOS**: Safari 13.0+ (рекомендуется установка как PWA)
- **Android**: Chrome 70+
- **HTTPS**: обязательно для Web Bluetooth API

### 📱 Установка как PWA (рекомендуется)

#### iPhone/iPad
1. Открыть Safari и перейти на сайт приложения
2. Нажать кнопку "Поделиться" (📤)
3. Прокрутить вниз и выбрать "На экран «Домой»"
4. Нажать "Добавить"
5. Приложение появится на домашнем экране как нативное приложение

#### Android
1. Открыть Chrome и перейти на сайт приложения
2. Нажать кнопку меню (⋮) в правом верхнем углу
3. Выбрать "Установить приложение" или "Добавить на главный экран"
4. Подтвердить установку

### 🏃‍♂️ Быстрый старт
1. Скачать все файлы проекта
2. Разместить на HTTPS сервере
3. Открыть в браузере и установить как PWA
4. Запустить приложение с домашнего экрана
5. Нажать "🎯 Кубик отзовись"

### ✨ Преимущества PWA
- **Офлайн работа** - приложение работает без интернета
- **Быстрый запуск** - мгновенная загрузка с домашнего экрана
- **Полноэкранный режим** - без интерфейса браузера

## 🔒 Безопасность

### 🛡️ Меры безопасности
- **HTTPS только** - Web Bluetooth требует безопасное соединение
- **User Gesture** - подключение только по действию пользователя
- **Валидация данных** - проверка входящих данных от кубика
- **Ограниченный доступ** - работа только с конкретным UUID

### 🔒 Приватность
- **Локальное хранение** - данные не покидают устройство
- **Без отслеживания** - нет аналитики или телеметрии
- **Минимальные права** - запрашивается только BLE доступ
